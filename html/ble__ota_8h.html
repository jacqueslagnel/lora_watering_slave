Detailed Description

Interface for the BLE OTA mode used by the slave firmware.

This file declares the functions used to request a boot into BLE OTA update mode, detect that request at startup, and run the OTA service loop. Accepted OTA durations are bounded by constants exposed in the BleOta namespace.
